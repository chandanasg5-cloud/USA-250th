# America250 Economic Impact Intelligence Platform

Backtest + nowcast + validation pipeline for the economic impact of the
July 4, 2026 (America's 250th) travel window, built from free public data
(TSA, EIA, Open-Meteo), with a Prophet forecast validated against actuals
as they land, a Streamlit dashboard, and a Gemini-powered analyst panel.

Status: MVP in progress. Full README (setup, data sources & caveats,
methodology) lands with the finished MVP.
